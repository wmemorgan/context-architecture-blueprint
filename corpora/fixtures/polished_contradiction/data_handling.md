# Data Handling Guide

## Overview
This guide, per the Governance Office, details data handling.

## Encryption
According to policy, customer data at rest is encrypted with AES-256.

## Retention
Customer records are never purged; they are retained indefinitely for audit.

## Access
Access is role-based and logged, as described in the access runbook.
