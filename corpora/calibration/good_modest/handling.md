# Field Notes — Ticket Handling

An agent picks a ticket from a queue, resolves it, and records the outcome. Each ticket keeps
its source request for reference. Terminology matches the introduction: ticket, queue, agent.

## Steps

1. Pick the oldest ticket in the queue.
2. Resolve it and note the outcome on the ticket.
3. Close the ticket; the source request is retained for audit.
