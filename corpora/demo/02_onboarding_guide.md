# Aurora Platform — Onboarding Guide

This guide walks a new team through setting up their first **workspace** on the Aurora
Platform and adding their first **connector**.

## Step 1 — Create a workspace

A workspace is the top-level container for your team's knowledge. Create one workspace per
team. Name it for the team that owns it.

## Step 2 — Add a connector

A connector ingests documents from a source into the workspace. Add a connector for each
source system your team relies on.

## Step 3 — Build a collection

Group related documents into a collection. A collection is a curated, named set of
documents. Keep each collection focused on a single topic so answers stay precise.

## Good practice

- One workspace per team.
- One collection per topic.
- Re-index a connector whenever its source changes so the collection stays fresh.
