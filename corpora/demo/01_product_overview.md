# Aurora Platform — Product Overview

The **Aurora Platform** is a workspace for teams to connect, organize, and query their
operational knowledge. A *workspace* is the top-level container; each workspace holds
*connectors* (sources of documents) and *collections* (curated sets of documents).

## Core concepts

- **Workspace** — the top-level container for a team's knowledge.
- **Connector** — an integration that ingests documents from a source.
- **Collection** — a curated, named set of documents within a workspace.

## Who uses Aurora

Operations, support, and knowledge teams use the Aurora Platform to keep a single,
consistent answer to recurring questions. Every answer traces back to a source document
in a collection.

## Versioning

Each collection is versioned. When a document changes, the collection version increments
and the prior version remains retrievable for audit.
